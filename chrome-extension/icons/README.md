## Regenerating Icons

To regenerate icons from the source image, run:
```bash
cd chrome-extension
./generate-icons.sh
```

The script uses macOS built-in `sips` tool or ImageMagick (if installed) to resize the source image to all required sizes.

## Icon Guidelines:
- Use PNG format
- Keep designs simple and recognizable at small sizes
- Use the Reddzit brand colors
- Consider using the Reddit alien or "r" symbol with a distinctive Reddzit twist

## Temporary Solution:
You can use online tools like:
- https://www.iconarchive.com/
- https://www.flaticon.com/
- Create simple icons with text "R" for Reddzit
